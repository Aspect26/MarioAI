NetworkSettings

data class NetworkSettings : Serializable

Settings of an artificial neural network which controls Super Mario agents.

Parameters

receptiveFieldSizeRow - number of rows in agent's receptive field size.

receptiveFieldSizeColumn - number of columns in agent's receptive field size.

receptiveFieldRowOffset - row offset of agent's receptive field. The default agent's location is in the middle of the receptive field.

receptiveFieldColumnOffset - column offset of agent's receptive field. The default agent's location is in the middle of the receptive field.

hiddenLayerSize - number of neurons in the hidden layer.

denseInput - specifies, whether the input grid tiles should be split in a half resulting in more dense grid around the agent being provided to the network.

oneHotOnEnemies - specifies, whether the enemies should be encoded using one hot encoding.
