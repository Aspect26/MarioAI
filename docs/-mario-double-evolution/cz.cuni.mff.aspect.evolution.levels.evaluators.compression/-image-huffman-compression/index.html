ImageHuffmanCompression

class ImageHuffmanCompression : ImageCompression

Compression of an image using adjusted Huffman compression.

The image is split into 2x2 pixels wide grid, and each tile gets its own character. The same tiles will get the same characters. The image is then converted to a String by going through the tiles from left to right, from top to bottom and taking each tile's character. This string is then compressed using original Huffman algorithm. This way, the more tiles in the image are different, the higher the resulting compression size will be.
