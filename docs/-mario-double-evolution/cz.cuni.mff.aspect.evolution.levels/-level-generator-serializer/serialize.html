serialize

fun serialize(levelGenerator: LevelGenerator): String

Serializes the given level generator to its string representation if the generator is supported. Supported level generators are cz.cuni.mff.aspect.evolution.levels.pmp.PMPLevelGenerator and cz.cuni.mff.aspect.evolution.levels.chunks.PCLevelGenerator. The generator can be then deserialized using deserialize.

Parameters

levelGenerator - the generator to be serialized.
