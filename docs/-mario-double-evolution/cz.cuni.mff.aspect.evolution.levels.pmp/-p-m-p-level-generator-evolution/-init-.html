PMPLevelGeneratorEvolution(populationSize: Int, generationsCount: Int, fitnessFunction: PMPLevelEvaluator<Float>, objectiveFunction: PMPLevelEvaluator<Float>, alterers: Array<Alterer<DoubleGene, Float>> = arrayOf(UpdatedGaussianMutator(0.2, 0.1)), evaluateOnLevelsCount: Int = DEFAULT_EVALUATE_ON_LEVELS_COUNT, levelLength: Int = DEFAULT_LEVEL_LENGTH, chartLabel: String = DEFAULT_CHART_LABEL, displayChart: Boolean = true)

Implementation of a Super Mario level generator evolution of PMPLevelGenerator. The implementation is highly customizable in terms of multiple properties of the evolution, which can be specified via primary constructor.

Parameters

populationSize - population size during the evolution.

generationsCount - number of generations of the evolution.

fitnessFunction - function computing fitness value of individuals.

objectiveFunction - function computing objective value of individuals.

evaluateOnLevelsCount - number specifying on how many levels, generated by a generator represented by some individual the fitness and objective functions should be computed.

levelLength - level lengths property of PMPLevelGenerator.

chartLabel - label of the evolution's chart.

displayChart - specifies, whether the evolution's chart should be displayed in realtime.
