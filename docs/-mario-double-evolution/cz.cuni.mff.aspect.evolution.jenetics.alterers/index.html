MarkovChainMutator

Implementation of a custom Markov Chain mutator. It does 2 mutations in the Markov Chain: swap 2 probabilities in one random variable of the MC; change a random probability in a random random variable, adjusting the other probabilities in that random variable so it is still valid after the operation.
