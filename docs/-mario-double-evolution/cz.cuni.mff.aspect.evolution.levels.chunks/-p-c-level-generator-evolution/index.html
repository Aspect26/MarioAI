PCLevelGeneratorEvolution

class PCLevelGeneratorEvolution : JeneticsLevelGeneratorEvolution<PCLevelGenerator>

Implementation of a Super Mario level generator evolution using PCLevelGenerator algorithm for level generation. The implementation is highly customizable in terms of multiple properties of the evolution, which can be specified via primary constructor.

Parameters

populationSize - population size during the evolution.

generationsCount - number of generations of the evolution.

fitnessFunction - function computing fitness value of individuals.

objectiveFunction - function computing objective value of individuals.

evaluateOnLevelsCount - number specifying on how many levels, generated by a generator represented by some individual the fitness and objective functions should be computed.

chartLabel - label of the evolution's chart.

displayChart - specifies, whether the evolution's chart should be displayed in realtime.
