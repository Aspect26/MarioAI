Package cz.cuni.mff.aspect.evolution.jenetics.evaluators

Contains implementations of custom evaluators for Jenetics library.

Types

MarioJeneticsEvaluator

This evaluator evaluates also objective value, not only fitness and has the ability to evaluate fitness of each individual in each generation whereas the default Jenetics evaluator evaluates fitness of an individual only once throughout the whole evolution.
