createMinified

fun createMinified(level: MarioLevel, noAlpha: Boolean = false): BufferedImage

Creates minified image from given Super Mario level, where each tile is represented by one pixel. Each tile uses its own color from custom color palette.

Parameters

level - the level's whose image is to be created.

noAlpha - specifies, whether the resulting image should contain alpha channel.
